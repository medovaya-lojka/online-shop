# JS store

Интернет-магазин одежды написанный на JS, Express.js, включает в себя админ-панель, интерфейс покупателя. БД хранит в json-файле.

## Запуск
1. Скачать репозиторий
2. Выполнить `npm install`
3. Выполнить `npm run dev`
